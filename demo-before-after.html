03 — Anti-Pattern Analysis

What /audit would flag on the "before" output

BEFORE — /audit would flag 8 violations

✕
Centered hero — DESIGN_VARIANCE=8 requires asymmetric layout. Hero H1 is center-aligned on a marketing page.
✕
Saturated gradient — #7c3aed → #2563eb violates the "no neon/saturated gradients" rule. Saturation > 80%.
✕
Round stats — "50% fewer bugs", "3x faster", "10k+ users". Organic numbers required (47.2%, 3.1×, 2,847 teams).
✕
Emojis in UI — 🤖🔒⚡🔧📊⚙️ are banned in production UI copy. Use SVG icons with consistent stroke weight.
✕
Vague CTAs — "Get Started Free" and "Learn More" provide no context. Requires specificity: "Start free trial" / "Watch 2-min demo".
✕
Equal feature cards — 6 identical cards with no visual hierarchy. Bento grid required: card size must map to feature importance.
✕
Generic social proof — "10,000+ Users", "⭐ 4.9/5 Rating". No specificity. Requires named teams and concrete metrics.
✕
Purple accent everywhere — Accent color used for backgrounds, borders, gradients, stats. 60-30-10 rule violated.

AFTER — /audit passes all checks

✓
Asymmetric hero — 50/50 grid, headline left-aligned. Respects DESIGN_VARIANCE=8.
✓
Single accent — #34d399 emerald used only for emphasis: CTA, stats highlights, micro-dots. Saturation < 60%.
✓
Organic numbers — 47.2%, 3.1×, 2,847 teams, 12 min avg. All with sourced deltas ("↓ from 41 min").
✓
SVG icon system — Consistent 16px icons, 1.2px stroke, matching accent color per category.
✓
Specific CTAs — "Start free trial" + arrow, "Watch 2-min demo" + play icon. Both have clear value propositions.
✓
Bento hierarchy — AI Review (3-col), Security (2-col), Analytics (4-col). Size signals importance.
✓
Named social proof — Avatar initials + "2,847 engineering teams". Specific and credible.
✓
WCAG AA contrast — All text/background pairs verified. #a1a1aa on #09090b = 5.9:1.

04 — The 8 Slash Commands

Each command loads the right references automatically — no manual context needed

/build
Generate a full page from scratch — dark-mode premium by default. Specify page type and description.
/polish
Final pre-ship pass. Tightens contrast, spacing, copy, micro-interactions. Non-destructive.
/audit
WCAG 2.2 AA + anti-slop report. Outputs file:line violations. No edits.
/critique
UX design review as a principal designer. Nielsen 10 heuristics, cognitive load analysis. No edits.
/animate
Add motion patterns — perpetual micro-interactions, spring physics. Respects prefers-reduced-motion.
/imagify
Gemini pipeline — replace placeholders with cinematic AI images. Falls back to curated picsum.
/dials
Adjust the 3 design dials live: variance=N motion=N density=N. Rebalances the output.
/variant
Swap brand preset — re-theme colors, typography, spacing, motion. 7 presets: vercel, linear, stripe…
